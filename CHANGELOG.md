# Registro de Cambios

Todos los cambios notables de este proyecto se documentan en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es/1.0.0/),
y este proyecto sigue [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2025-06-01

### Agregado
- 🎉 Lanzamiento inicial de MiBolsillo
- Dashboard mensual con balance, ingresos y egresos en tiempo real
- Registro de movimientos con campos: título, monto, categoría, fecha y notas
- 10 categorías fijas con íconos y colores únicos
- Soporte para tipos de movimiento: ingreso y egreso
- Filtros por tipo y por categoría con chips interactivos
- Formulario de creación y edición de movimientos con validaciones
- Eliminación de movimientos con diálogo de confirmación (long press)
- Base de datos local SQLite con patrón Singleton
- Resumen financiero del mes actual mediante consultas SQL
- Diseño Material 3 con paleta morado + azul + blanco
- Soporte para Android e iOS
- Localización en español
